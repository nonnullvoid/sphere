# Layer: Public Persona

Your intentional public presence — the intersection of who you genuinely are and what you want to project. Left untended, public persona fills in with noise and default assumptions. This layer makes it deliberate.

**Works with:** LifeSphere

---

## What lives here

```
layers/public-persona/
├── README.md           ← this file
├── identity.md         ← your archetype, tone, what you're building toward
├── voice.md            ← how you write and speak across contexts
├── style.md            ← visual system — colors, type, aesthetic
└── platforms/
    ├── index.md        ← handle strategy and footprint overview
    ├── github.md       ← repos, bio, what to pin
    ├── x.md            ← handle, bio, what to post
    ├── linkedin.md     ← headline, about, how it differs
    ├── website.md      ← personal site structure and strategy
    └── blog.md         ← platform, cadence, topics
```

---

## When the AI loads this layer

Load when:
- Writing a bio, README, or any public-facing copy
- Deciding whether/how to publish something
- Reviewing tone on any public output
- Planning a new project or platform presence

---

## Install

```bash
cp -r layers/public-persona your-sphere/layers/public-persona
```

Add to `CLAUDE.md`:
```markdown
- `layers/public-persona/` — when writing public-facing copy, bios, or deciding how to present work
```
